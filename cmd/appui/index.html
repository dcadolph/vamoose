What is vamoose

A smart calendar-workflow engine. Declare ordered steps that branch, wait, and gate on approval, and vamoose runs them across any calendar backend from your terminal, Claude, or Slack. Time off is the flagship workflow, not the whole scope.

Getting started

  1. Install: brew install dcadolph/tap/vamoose
  2. Sign in to your calendar with vamoose login, or set your provider on the Setup page.
  3. Request time off: vamoose off next week --watch
  4. Advance approvals in the background: vamoose daemon
  5. Watch and act on it right here in the dashboard.

Quick commands

  • vamoose off next week request time off from a phrase
  • vamoose run <workflow> next week --watch run any workflow and let the daemon advance it
  • vamoose check see whether the manager approved
  • vamoose daemon advance watched holds in the background
  • vamoose history the run log, also on the Runs page here
  • vamoose app this dashboard

Workflow verbs

hold, approve, notify, note, away, event, cancel, message, wait, leave. An approve step can branch on the outcome, gate on a timeout, and chain several approvers.

Backends

Microsoft Graph, Google Calendar, Apple iCloud, and any CalDAV host, behind one interface. Pick with --provider or VAMOOSE_PROVIDER. See the Setup page for what is configured.
